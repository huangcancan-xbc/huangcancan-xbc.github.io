👋 欢迎来到我的个人博客

这里是我记录学习心得、技术分享与生活点滴的小天地。希望这些内容能为你提供实用价值，一起在技术道路上成长！ 🙌

🌐 关于网站访问

因为我的这个主域名是在华为云买的，要备案需要填写互联网信息服务等相关信息，一方面个人感觉备案流程繁琐且复杂 😡，另一方面是 CF 等一系列大厂服务给的比国内更多更香，更照顾我们这些个体开发者，没有理由不选择啊。

其次我个人认为技术无国界，想要一点技术言论自由，技术的应用与传播不应该受到限制，未来我会想要发一些好玩的内容，比如 VPS、网络知识科普等相关内容，这也是我决定全面拥抱 Cloudflare 的原因，剩下还有一些不太重要的原因就不一一叙述了……

在换了多次部署后，最终确定（应该不会再变了）：截至时间 2026/01/01，域名 NS 已完全托管至 Cloudflare，全站启用 Cloudflare 的 CDN 服务，为获得最佳访问体验，强烈建议科学上网，国内直连可能存在延迟或不稳定情况！后续也会持续对网站进行各方面的优化，感谢理解呀～🤪，好了，下面来具体看看部署细节：

  • minbit.top： 使用 Vercel 部署，函数执行区域默认 亚太节点，中国大陆部分地区访问速度较优。
  • huangcancan-xbc.github.io： GitHub Pages 原生部署，永久有效且无访问限制，核心优势是 稳定性拉满，需确保能 稳定访问 GitHub。
  • myblog-pi4waofd.edgeone.app： 使用 Edgeone pages 部署，说实话，这个域名太长不好记，都准备删除部署了，想着万一有什么用呢，于是就留下来了，先暂时留着吧，日后删除了会在这里更新。

域名续费提醒：minbit.top 域名将于 2027 年 5 月到期。目前计划续费，若有变动会及时在此页面更新。如 minbit.top 系列域名出现异常，请优先切换至 GitHub Pages 地址。

🧩 内容与资源说明

🎉🎉🎉 截至 2025.10.01 日，我已将本站图片迁移至 小米里的大麦图床，网站展示部分的图片均采用了新图床方案，但是为了保证长期稳定，博客文章内的图片依旧使用 GitHub + jsDelivr CDN，这对中国大陆的朋友做了优化，但是依旧推荐使用 科学上网 来达到最好的访问效果！

需要注意的是该图床每天提供 10 万次的请求，若本站遭到攻击或盗刷流量请求，依旧会导致对图片的请求失败！此时只能依靠浏览器缓存或者隔天访问！

  • 图片资源：展示页图片走小米里的大麦图床，文章内图片走 GitHub + jsDelivr（主要为了 长期稳定），但是不保证快速稳定地加载，延迟可能较高。所以 网站图片加载异常属于正常现象，可刷新重试，强烈建议科学上网进行解决！
  • 评论系统：采用 Powered by Twikoo v1.6.44 提供评论功能，需在良好网络环境下使用，避免评论加载失败或延迟。
  • 博客内容：核心聚焦后端研发方向，以 C++ 编程、Linux 系统应用为核心，辅以技术资源分享与生活杂谈；未来计划拓展前端（HTML/CSS/Vue）、数据库（MySQL/Redis）等相关领域内容，形成完整技术体系。
  • 算法相关：暂未开设算法相关博客，但已整理大量练习题与解析，欢迎查看：GitHub 的 Algorithm 仓库 或者 Gitee 的 Algorithm 仓库，这里包含了算法原理、难点解析、AC 源代码等。
  • 更新频率： 因处于大学阶段，需兼顾学业与多平台维护，因此更新以 月为单位 为主，也巧于“研究一代、成熟一代、发布一代”，后续会根据时间精力优化更新节奏，敬请期待。

博客内容源于我的真实学习笔记与实战经验，力求还原技术探索的完整过程，但文字表达难免有局限，若部分知识点有描述不清之处，欢迎通过评论或邮件反馈和指正，我会及时补充和优化 😊！

📢 网站使用建议

  1. 网络环境：优先选择 科学上网！搭配 Chrome、Edge 等现代浏览器，可显著提升页面加载速度与功能兼容性！
  2. 学习方法：博客内容按技术体系分类组织，建议从基础专栏开始，按顺序循序渐进学习，结合目录导航快速定位目标内容。
  3. 实践建议：技术学习的核心是动手实践，“实践才是检验真理的唯一标准”，建议跟随文章内容逐行编写代码、调试环境，遇到问题先自主排查，再参考相关资源。
  4. 反馈交流：欢迎通过评论系统或者下方邮件联系我，看到会及时回复，欢迎分享你的学习心得和疑问，期待与你共同进步！

📌 关于我

我是一名计科专业的在校大学生，也是一名热爱技术的学习者和分享者。2023 年 8 月踏入大学校园，从最初的迷茫到如今对技术的深度热情，一路在探索中成长，这段旅程让我收获颇多。于我而言，技术学习不仅是谋生技能，更是思维训练和解决问题的能力培养。

这个博客记录了我的学习笔记、实战心得、生活杂谈等，我希望将自己的学习历程与踩坑经验沉淀下来，帮助更多同行者少走弯路、更高效成长。虽然文字无法完全还原实践中的细节与思考，但我会持续打磨内容和优化内容质量，让分享更有价值！
