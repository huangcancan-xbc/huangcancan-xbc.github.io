💡 主要技能概览

C/C++ 编程

熟练掌握 C/C++ 语法与面向对象编程，具备多线程、泛型编程能力；熟悉 C++11 现代特性及 RAII 资源管理，能够编写安全高效代码，能实现 STL 核心组件简化版本。

数据结构与算法

熟悉常用数据结构及图相关算法，能够运用双指针、二分、哈希、BFS/DFS、动态规划、贪心等多种策略解决复杂问题；了解高级检索结构如布隆过滤器与 AVL 树的设计原理。

Linux 开发

熟练掌握 Linux 环境下的开发工具链，包括常用命令、Git 版本控制、GCC 编译器、Vim 编辑器的使用及 Makefile 编写，并深入理解进程/线程模型，能够进行多进程/线程编程与进程间通信开发。

网络编程

了解 TCP/IP 网络模型及 TCP/UDP/HTTP 等协议，具备 Socket 网络编程能力，熟悉五种高级 I/O 模型与 select/poll/epoll 多路复用技术。

数据库

能使用 MySQL 完成基础的增删查改（CRUD） 操作。

Qt 开发

熟练使用 QtCreator 进行 GUI 应用开发；深入理解信号槽机制与事件处理体系，掌握 QWidget 控件与图形绘制；能运用 QSS 样式表完成界面美化，了解 QML。

其他

  • 掌握 Python 基础及 HTML/CSS/JS 前端基础，未来期望学习：Rust ≥ Go > TS。
  • 性格沉稳，热爱编程，对技术有好奇心。持续关注 AI 与大模型等前沿动态。
  • 对 codex、opencode、openclaw、trae、、gemini cli、、cursor、claude code 等工具有所了解和涉及。
