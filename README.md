# 🎨 我的 Hexo 博客（Butterfly 主题）

欢迎访问我的个人博客！本博客基于 [Hexo](https://hexo.io/) 和 [Butterfly](https://butterfly.js.org/) 主题搭建，并托管在 GitHub Pages 上，支持个性化定制和插件扩展。

## 🚀 项目概述

本仓库包含我的博客源码，使用 Hexo 生成静态网页，并通过 GitHub Pages 部署，让你可以轻松搭建一个美观、高效的个人博客。

## 📌 主要特性

- ✍️ **基于 Hexo**：轻量、快速的静态博客框架
- 🎨 **使用 Butterfly 主题**：美观且高度可定制
- 🌍 **支持 GitHub Pages 部署**：便捷发布，无需服务器
- 🔌 **多插件扩展**：增强功能，如搜索、评论、统计等

## 📖 安装与使用

### 1️⃣ 克隆仓库

```bash
git clone https://github.com/huangcancan-xbc/huangcancan-xbc.github.io.git
cd huangcancan-xbc.github.io
```

### 2️⃣ 安装依赖

```bash
npm install
```

### 3️⃣ 本地运行

```bash
hexo server
```

然后在浏览器中访问 [`http://localhost:4000`](http://localhost:4000) 预览博客。

### 4️⃣ 生成并部署

```bash
hexo clean && hexo generate && hexo deploy
```

## 🔧 配置

你可以根据需求修改以下配置文件：

- **博客基础配置**：`_config.yml`
- **主题个性化配置**：`themes/butterfly/_config.yml`
